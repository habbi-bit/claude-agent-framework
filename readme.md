# Claude Agent Framework 🤖

Ein agentisches System in Python, das die Anthropic Claude-API nutzt. 

## 🚀 Features
* **Anthropic Tool Use:** Dynamische Workflow-Orchestrierung über Function Calling.
* **Prompt Caching:** Kosten- und Latenzoptimierung über Ephemeral Breakpoints.

## 🛠️ Setup
* Python 3.11+
* Anthropic Python SDK
